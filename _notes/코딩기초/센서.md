#### 저항 값을 알면 주변 환경을 알 수 있다.

R1보다 R2가 매우 크면
Vcc 보다는 작고 Vout
0<Vout < Vcc  (전압)을 ad컨버터를 통해 디지털화시킨다. 
분해능에 따라 8비트면 256가지, 10비트면
0<= D(디지털값) <= 1023 까지가 나올 수 있는것이다.

Vout 은 x로 잡고 (x의 최대는 Vcc, 따라서 5V) 
x 5v = D : 1023
x= 5 (곱하기) D / 1023
ex) D가 512 면 2.5v 나옴


따라서 
2.5v = R2 / R1+R2  ) x (5v)
R1은 이미 우리가 넣은거라서 알고 있음
따라서 R2 = 값 나옴
그래서 저항이 나왔으니, 온도를 알 수 있음.
(R1 = 최대값 - R2)
