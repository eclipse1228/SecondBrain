

C++에서는 문자열을 하나의 변수 type처럼 사용하며, 문자열을 훨씬 다양하고 쉽게 다룰 수 있게 해 준다. 

- char* , char[] 과 다르게 **문자열의 끝에 '\0' 문자가 들어가지 않으며, 문자열의 길이를 동적으로 변경 가능**하다. 

- cin , cout , getline 함수 사용 가능

**cin >> str;**          // 공백(space) 이전까지의 문자열을 입력 받는다.
